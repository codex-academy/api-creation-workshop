# Heroku deploy

## Git setup

* `touch .gitignore`
*  Add `node_modules/` to the `.gitignore`


* `git init`
* `git add .`
* `git commit -m "initial commit"`
* `git brnach -M main`


## Heroku setup

* `heroku create`
* `heroku apps:rename put-your-app-name-here`
* `git push heroku main`
